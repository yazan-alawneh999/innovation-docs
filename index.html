3. System Architecture

The application follows a client-server architecture where multiple frontends (Android & Web) communicate with the backend via RESTful APIs.

Key Components:

  • Frontend: Android (MVVM) & Web (Vue 3/Nuxt.js)
  • Backend: Spring Boot Microservices (Java 17), OAuth2 Security
  • Infrastructure: Docker, Google Cloud, Docker Compose Secrets
